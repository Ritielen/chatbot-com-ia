🤖 Chatbot com IA e Python

Projeto desenvolvido durante a Jornada Python da Hashtag Treinamentos. O objetivo foi criar um sistema de respostas automatizadas utilizando a API da OpenAI, capaz de manter o contexto das conversas e interagir de forma inteligente.

🚀 Funcionalidades

Geração de Respostas: Integração com modelos de linguagem da OpenAI.

Gestão de Contexto: O chatbot "lembra" o que foi dito durante a conversa.

Interface Web: Criada com Streamlit para uma experiência de usuário fluida e moderna.

🛠️ Tecnologias Utilizadas
Python 🐍

Streamlit (Interface do Usuário) 💻

OpenAI API (Inteligência Artificial) 🧠
